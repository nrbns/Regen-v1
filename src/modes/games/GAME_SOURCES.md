## Legal Notes

1. **Always Check License**: Even MIT-licensed games may have asset restrictions
2. **Attribution Required**: Most open-source games require attribution
3. **Commercial Use**: Verify license allows commercial use
4. **No Proprietary Content**: Don't use games from Friv, CoolMathGames, etc. without permission
5. **Contact Publishers**: For embeddable games, always contact publisher first

## Quick Checklist

Before adding a game:
- [ ] Verified license is permissive (MIT, Apache 2.0, etc.)
- [ ] Checked for attribution requirements
- [ ] Confirmed commercial use is allowed
- [ ] Added source link to catalog
- [ ] Included proper attribution
- [ ] Tested game loads correctly
- [ ] Verified offline capability (if marked)
