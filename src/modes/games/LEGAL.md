# Legal & Licensing Information

## Overview

This Game Hub uses **only legally obtainable games** from open-source repositories and properly licensed sources. We do **not** copy or redistribute proprietary games from Friv or other commercial game portals.

## Legal Sources

### Attribution Requirements

Each game includes:

- **License**: License type (MIT, etc.)
- **Source**: Link to original repository

## Adding New Games

When adding games to the catalog:

1. **Verify License**: Ensure the game has a permissive license (MIT, Apache 2.0, etc.)
2. **Check Attribution**: Include proper attribution in the game entry
3. **Source Link**: Always include a link to the original source
4. **Commercial Use**: For commercial use, verify the license allows it
5. **No Proprietary Content**: Do not include games from Friv, Kongregate, or other proprietary portals without explicit permission

## Embedding Third-Party Games

If you want to embed games from third-party portals:

1. **Contact Publisher**: Reach out to CrazyGames, Kongregate, or HTML5Games.com
2. **Request Permission**: Ask for embed/API terms
3. **Review Terms**: Ensure terms allow embedding in your application
4. **Document License**: Add license information to the game catalog entry
5. **Comply with Terms**: Follow any attribution or revenue share requirements

## DMCA & Takedown

If you receive a DMCA takedown notice:

1. Remove the game immediately
2. Document the request
3. Contact the game developer if possible
4. Update the catalog to reflect removal

## Sample Licensing Template

When contacting game publishers for embedding rights:

```
Subject: Request for Embedding Permission - [Game Name]

Dear [Publisher/Developer],

I am developing a browser-based game hub (similar to Friv) and would like to 
request permission to embed [Game Name] in our platform.

Our platform:
- Provides free access to games
- Includes proper attribution
- Links back to original source
- Does not modify game code

Please let me know:
1. If embedding is permitted
2. Any attribution requirements
3. Revenue share or licensing fees (if applicable)
4. Terms and conditions

Thank you for your consideration.

Best regards,
[Your Name]
```
