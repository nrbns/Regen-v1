## Troubleshooting

### Regen Not Responding

- Check if engine is running (`http://localhost:3030/health`)
- Check browser console for errors
- Try reloading the page

### Voice Not Working

- Check microphone permissions
- Ensure browser supports Web Speech API
- Try different browser/OS

### Commands Not Executing

- Check if tab is active
- Verify IPC handlers are registered
- Check console for errors

### Language Detection Wrong

- Language detection uses Unicode scripts
- Mixed languages may default to English
- Can manually set language in future
