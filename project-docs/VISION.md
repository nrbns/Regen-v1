## Technical Vision: Architecture Principles

### 1. Event-Driven by Default

All user actions flow through an event bus, enabling real-time pattern detection without polling or scanning.

### 2. Local-First Architecture

* Default: All processing happens locally
* Optional: Cloud sync only if user explicitly enables
* Privacy: User data never leaves device without consent

### 3. Explicit Consent for AI Actions

Every AI action requires user permission. No autonomous behavior without consent.

### 4. Transparent Execution

All AI actions are logged and visible to the user. No hidden processes.

### 5. Graceful Degradation

Regen works perfectly as a browser even if AI features are disabled or offline.
