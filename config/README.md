# Configuration Files

This directory contains all configuration files for the Regen Browser project.

- **Build Configs**: `vite.config.ts`, `vite.config.redix.ts`, `vitest.config.ts`, `vitest.setup.ts`
- **Linting/Formatting**: `eslint.config.mjs`, `.prettierrc.json`, `.prettierignore`, `lint-staged.config.js`
- **Testing**: `playwright.config.ts`, `playwright.visual.config.ts`
- **Styling**: `tailwind.config.ts`, `postcss.config.cjs`
- **Other**: `.lighthouserc.js`, `.releaserc.json`, `example.env`, `.env.realtime.example`, `requirements-worker.txt`
