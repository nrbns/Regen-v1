## AI Philosophy

Regen treats AI as a **tool**, not a controller.

- AI never runs automatically
- AI never interrupts browsing
- AI only responds when you ask
- AI runs locally when possible
- Browser works fully even if AI is unavailable

If AI fails, **the browser continues normally**.

---

## Offline & Privacy

- No mandatory accounts
- No forced sync
- No background tracking
- Local models supported (via Ollama or similar)
- Network use is explicit and user-driven

Your browsing remains **your business**.

## Quick Start

```bash
# Clone and install
git clone https://github.com/nrbns/Regenbrowser.git
cd Regenbrowser
npm install

# Start development server
npm run dev:web
```

Visit `http://localhost:5181` to use Regen.
