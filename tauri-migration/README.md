# Regen Tauri

This is the Tauri version of Regen, migrated from Electron.

## Setup

1. Install Rust: https://rustup.rs/
2. Install dependencies: `npm install`
3. Run dev: `npm run tauri dev`
4. Build: `npm run tauri build`
